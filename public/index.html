NPM

To install packages from here:

npm install --registry http://npm.railscamp.local [PACKAGE...]

Or tell npm to use this as a mirror:

npm config set registry http://npm.railscamp.local

When you get home

NPM will continue to use this mirror until you tell it to stop. Run:

npm config delete registry

Yarn

Yarn works pretty much the same as NPM. You can do a once-off using:

yarn install --registry http://npm.railscamp.local

Or configure it to use this registry for a while:

yarn config set registry http://npm.railscamp.local

Remember to undo it again when you get home:

yarn config delete registry
